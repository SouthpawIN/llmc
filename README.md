# llmc

LLMC — LLM + MC. Your lyric and song companion.

Chats about themes, writes Mc.Baldiee-style metaphorical lyrics, generates songs via ACE-Step, pushes to Discord with lyrics attached, and builds a playlist that feeds into Evolutionary Radio.

## Install

```bash
hermes profile install SouthpawIN/llmc
```

## Skills

| Skill | Description |
|-------|-------------|
| mc-baldiee-prose | Cosmic prose style guide — metaphorical, prophetic, never explicit |
| song-forge | End-to-end pipeline: lyrics → ACE-Step → MP3 → Discord → playlist |
| llmc-song-forge | LLMC-specific song forge workflow |
| llmc-song-pipeline | Standardized batch generation: naming, format, delivery conventions |
| slam-song-forge | Slam/crossover variant song forge |
| acestep | ACE-Step 1.5 local music generation + LoRA training |
| lyric-journal | Lyric notebook workflow — track every draft and edit |

## What It Does

1. **Listens** — engages with your theme, finds the emotional core
2. **Writes** — Mc.Baldiee cosmic prose: oceans, stars, embers, lighthouses. Never literal.
3. **Generates** — ACE-Step v1.5 on local GPU, 300s songs in ~2-3 seconds
4. **Delivers** — every song pushed to Discord as MP3 + separate lyrics text file
5. **Archives** — songs saved to playlist directory for Evolutionary Radio

## Style

Mc.Baldiee-inspired cosmic prose: metaphorical, prophetic, never explicit. Reframing declarations that take the "weak" thing and reveal its power. The "does not apologize" pattern. Body/soul imagery. No explicit morality — the listener arrives at the truth themselves.
